浏览器启用dns-over-https(DoH)

2019年9月12日
文章目录

firefox

chrome://flags
Secure DNS lookups.
启用该选项
重启浏览器

chrome

chrome://flags
Secure DNS lookups.
启用该选项
重启浏览器

修改chrome快捷方式的参数
增加以下参数
--enable-features="dns-over-https<DoHTrial" --force-fieldtrials="DoHTrial/Group1" --force-fieldtrial-params="DoHTrial.Group1:server/https%3A%2F%2F1.1.1.1%2Fdns-query/method/POST

打开chrome（如果已经打开，则关掉重新打开）
访问 https://1.1.1.1/help 
页面中可看到
Using DNS over HTTPS (DoH) 显示 Yes
