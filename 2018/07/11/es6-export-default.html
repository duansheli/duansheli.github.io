export 和 export default 的区别

2018-07-11 约 296 字 预计阅读 1 分钟

  • 相同与差异
  • 案例演示

区别

相同与差异

都可用来导出常量、函数、模块等
在一个模块中 export、import可以多次使用

export方式导出， 需要用{} export default 则不需要
导入模块时， 同理

在同一个模块中只能用一次 export default
